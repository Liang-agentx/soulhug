# 快速开始

3 步跑起来。

---

## Step 1：安装全局依赖

```bash
# 安装 pnpm（包管理器）
npm install -g pnpm

# 安装 PromptX（AI 工具框架）
npm install -g promptx
```

**验证安装：**

```bash
promptx --version
# 输出版本号，如：1.27.4
```

**PromptX 是什么？**

PromptX 通过 MCP（Model Context Protocol）协议，给 AI 提供：
- **角色能力** — 定义 AI 是谁
- **工具能力** — 让 AI 能调用外部 API

项目启动时，会自动运行 `promptx mcp-server`。

---

## Step 2：安装项目 & 配置

```bash
# 安装项目依赖
pnpm install

# 创建环境变量文件
cp .env.example .env
```

**编辑 `.env`：**

```env
# 必填：Anthropic API Key
ANTHROPIC_API_KEY=sk-ant-api03-xxxxx

# 可选：OKX 模拟盘交易
OKX_API_KEY=你的key
OKX_SECRET_KEY=你的secret
OKX_PASSPHRASE=你的passphrase
```

---

## Step 3：启动

```bash
pnpm dev:full
```

## 出问题了？

### `promptx: command not found`

PromptX 没装。

```bash
npm install -g promptx
```

如果用 nvm，确保在正确的 Node 版本下安装：
```bash
nvm use 20
npm install -g promptx
```

### 页面显示「连接中...」

后端没跑起来。检查终端报错。

常见原因：
1. `ANTHROPIC_API_KEY` 没填
2. PromptX 没装

### AI 不回复 / 工具调用失败

1. 检查 `.env` 里的 API Key
2. 看终端有没有报错
3. 确认 PromptX 版本 >= 1.27

### 左侧区域空白

正常。OKX/币安禁止 iframe 嵌入，主要功能在右边 AI 对话。
