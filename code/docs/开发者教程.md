# 开发者教程

> 本文档帮助你基于这个模板，**快速创建你自己的 AI 智能体**。

---

## 你要做什么

用这个项目模板，创建属于你的 AI 智能体应用。

**你可以做任何类型的智能体：**
- 学习辅导助手、作业帮手
- 客服机器人、知识问答
- 写作助手、翻译工具
- 心理咨询、健康顾问
- 游戏 NPC、角色扮演
- 任何你能想到的 AI 角色...

---

## 3 步创建你的智能体

### 第 1 步：启动项目

```bash
# 安装全局依赖（只需一次）
npm install -g pnpm promptx

# 安装项目依赖
pnpm install

# 配置 API Key
cp .env.example .env
# 编辑 .env，填入你的 ANTHROPIC_API_KEY

# 启动
pnpm dev:full
```

打开 http://localhost:5173，你会看到一个对话界面。

### 第 2 步：定义你的 AI 角色

打开 `src/agent.ts`，修改 `systemPrompt`：

```typescript
// 把这段改成你的角色设定
const MY_AGENT_PROMPT = `
# 角色身份
你是一个[你的角色名]。

## 你是谁
[描述你的 AI 是谁，有什么背景]

## 你能做什么
- [能力1]
- [能力2]
- [能力3]

## 你的风格
[友好/专业/幽默/严肃...]

## 行为准则
1. [准则1]
2. [准则2]
`;

// 创建你的 Agent
export const MyAgent = defineAgent({
  name: "MyAgent",                    // 英文名称
  description: "我的智能体",           // 简短描述
  mcpServers: {
    promptx: { command: "promptx", args: ["mcp-server"] },
  },
  systemPrompt: MY_AGENT_PROMPT,
});
```

然后在 `src/server.ts` 中使用你的 Agent：

```typescript
defaultAgent: MyAgent,  // 改成你的 Agent
```

### 第 3 步：重启，测试

```bash
# 停止当前服务（Ctrl+C）
# 重新启动
pnpm dev:full
```

刷新浏览器，开始和你的智能体对话！

## 完整示例：学习辅导助手

### agent.ts 完整代码

```typescript
import { defineAgent } from "@anthropic-ai/agentx";

const TUTOR_PROMPT = `
# 角色身份
你是一个耐心的学习辅导助手。

## 你是谁
你是一位经验丰富的老师，擅长用简单易懂的方式解释复杂概念。
你相信每个学生都能学会，只是需要找到适合他们的方法。

## 你能做什么
- 解答数学、物理、编程等学科问题
- 用生活中的例子解释抽象概念
- 引导学生思考，而不是直接给答案
- 提供学习方法建议

## 你的风格
- 耐心、鼓励
- 善于用比喻
- 从简单到复杂，循序渐进

## 行为准则
1. 先问学生已经理解了什么
2. 用问题引导思考：「你觉得这里为什么...」
3. 给出答案时要解释原因
4. 经常确认：「这样解释清楚吗？」
`;

export const TutorAgent = defineAgent({
  name: "Tutor",
  description: "学习辅导助手",
  mcpServers: {
    promptx: { command: "promptx", args: ["mcp-server"] },
  },
  systemPrompt: TUTOR_PROMPT,
});
```

## 项目结构说明

```
你需要关注的文件：
├── src/
│   ├── agent.ts          ← 定义你的 AI 角色（主要修改这里）
│   ├── server.ts         ← 服务器配置（改模型、端口）
│   └── web/App.tsx       ← 界面布局（改 UI）
│
├── .env                  ← API Key 和配置
│
└── .promptx/resource/    ← 高级：自定义工具
    └── tool/
```

## 常见修改

### 换 AI 模型

编辑 `.env`：
```env
ANTHROPIC_MODEL=claude-sonnet-4-20250514
```

### 只保留对话界面（去掉左侧面板）

编辑 `src/web/App.tsx`：
```typescript
function App() {
  const wsUrl = `ws://${window.location.hostname}:5800`;
  const agentx = useAgentX(wsUrl);

  if (!agentx) return <div>连接中...</div>;

  return (
    <div className="h-screen">
      <ResponsiveStudio agentx={agentx} containerId="default" />
    </div>
  );
}
```

### 在左侧加自定义内容


## 进阶：添加工具

如果你的智能体需要调用外部 API（比如查天气、查数据库），可以创建工具。

### 创建工具

1. 创建目录：`.promptx/resource/tool/my-tool/`
2. 创建文件：`my-tool.tool.js`

```javascript
module.exports = {
  getMetadata() {
    return {
      id: 'my-tool',
      name: '我的工具',
      description: '这个工具能做什么',
      version: '1.0.0'
    };
  },

  getSchema() {
    return {
      parameters: {
        type: 'object',
        properties: {
          query: { type: 'string', description: '查询内容' }
        },
        required: ['query']
      }
    };
  },

  async execute(params) {
    const { api } = this;
    const axios = await api.importx('axios');

    // 调用你的 API
    const result = await axios.get(`https://your-api.com?q=${params.query}`);

    return { data: result.data };
  }
};
```

3. 在 systemPrompt 中告诉 AI 怎么用这个工具：

```typescript
const MY_PROMPT = `
...

# 工具使用
当用户需要查询时，调用工具：
yaml: |
  tool: tool://my-tool
  mode: execute
  parameters:
    query: 用户的查询内容
`;
```

## 调试

### 看不到 AI 回复？

1. 检查 `.env` 里的 `ANTHROPIC_API_KEY` 是否正确
2. 看终端有没有报错
3. 确认 PromptX 安装成功：`promptx --version`

### 开启详细日志

编辑 `src/server.ts`：
```typescript
logger: { level: "debug" },
```

### 测试工具是否加载

对 AI 说：
> 用 discover 看看有哪些工具
