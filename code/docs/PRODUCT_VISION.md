# SoulHug 产品愿景与体验设计

> **产品哲学**：理解是爱的另一种表达方式
> **产品定位**：帮助青少年理解父母视角的情感陪伴智能体

---

## 一、核心洞察

### 用户痛点
1. **说不出口** - 和父母吵架后，想道歉但拉不下脸
2. **不被理解** - 感觉父母只会批评，不懂自己
3. **无处倾诉** - 朋友不一定理解，心理咨询太严肃

### 独特价值
1. **角色扮演** - 让用户"听到"父母内心的声音
2. **心理活动外化** - 用括号展示父母的真实想法 `（其实我是担心你...）`
3. **具体可行** - 给出具体的沟通话术，而不是空洞的"多沟通"

---

## 二、视觉体验设计

### 当前问题
- UI 过于简单，缺乏情感温度
- 没有视觉反馈来强化情感连接
- 对话界面与普通聊天 App 无差异

### 🎨 设计方向：情感可视化

#### 1. 呼吸式核心视觉 (Breathing Core)

一个柔和的、有生命感的视觉元素：

```
┌─────────────────────────────────────┐
│                                     │
│          ╭─────────────╮            │
│       ╭──│    🤗       │──╮         │
│      ╭───│  (呼吸动画)  │───╮        │
│     ╭────│   ↕ 缩放    │────╮       │
│     ╰────│    glow     │────╯       │
│      ╰───│             │───╯        │
│       ╰──│             │──╯         │
│          ╰─────────────╯            │
│                                     │
│     "我在这里，听你说"              │
│                                     │
└─────────────────────────────────────┘
```

**技术实现**：
- CSS `transform: scale()` + `animation`
- 背景 radial-gradient 呼吸效果
- 可选：Three.js 粒子系统

#### 2. 情绪感知背景 (Emotion-Responsive Background)

根据对话情绪动态变化的背景：

| 情绪状态 | 背景效果 |
|---------|---------|
| 默认/平静 | 柔和的粉紫渐变，缓慢流动 |
| 倾诉中 | 温暖的橙粉色调，像被拥抱 |
| 愤怒/委屈 | 稍暗的紫色，但不压抑 |
| 理解后 | 明亮的天蓝+粉色，豁然开朗 |

**技术实现**：
- CSS 动态渐变 + transition
- 基于关键词的情绪检测
- 或使用 Claude 返回情绪标签

#### 3. 角色切换视觉 (Role Switch Animation)

当 SoulHug 切换到"父母角色"时：

```
普通对话                    父母角色扮演
    ╭───╮                     ╭───╮
    │🤗│   ──────────────>    │👨/👩│
    ╰───╯                     ╰───╯
 SoulHug                    扮演你的父母

 [过渡动画：渐变 + 轻微缩放 + 边框变化]
```

**视觉提示**：
- 头像变化（或头像上加面具效果）
- 气泡边框颜色变化
- 添加 "🎭 角色扮演中" 标签

#### 4. 心理活动气泡 (Thought Bubble)

将父母的心理活动用特殊视觉呈现：

```
┌─────────────────────────────────────────────┐
│ 👨 爸爸：                                    │
│ ┌─────────────────────────────────────────┐ │
│ │ 你怎么又玩手机！作业做完了吗！            │ │
│ └─────────────────────────────────────────┘ │
│                                             │
│     💭 ┌───────────────────────────┐       │
│        │ 其实我是担心你视力不好... │        │
│        │ 也怕你沉迷游戏耽误学习    │        │
│        └───────────────────────────┘        │
│        ↑ 心理活动（用虚线框 + 柔和背景）     │
└─────────────────────────────────────────────┘
```

#### 5. 情感时间线 (Emotional Journey)

对话结束时，展示情感变化轨迹：

```
你的情感旅程

愤怒  ●───╮
         │
委屈      ╰──●
              │       ╭──● 被理解
难过           ╰──●──╯

      开始   中间   现在
```

## 三、产品功能建议

### 核心功能（MVP）
1. ✅ 情感倾诉对话
2. ✅ 父母角色扮演
3. ✅ 心理活动展示
4. ⬜ 对话后反思总结

### 增强功能（V2）
1. **父母画像保存** - 记住用户父母的性格特点
2. **沟通话术生成** - 生成具体的和好话术
3. **情感日记** - 记录每次对话的情感变化
4. **成长轨迹** - 长期跟踪亲子关系改善

### 社区功能（V3）
1. **匿名故事分享** - "我和爸妈和好了"
2. **话术模板库** - 用户贡献的沟通话术
3. **家长端** - 让父母也能理解孩子视角（反向功能）

## 四、用户留存策略

### 1. 情感连接
- 记住用户的名字和父母信息
- 对话开头："上次你说和妈妈吵架了，后来怎么样了？"
- 定期关怀："最近和爸妈相处得怎么样？"

### 2. 成就系统
- "第一次尝试理解父母"
- "成功使用沟通话术"
- "连续 7 天没有和父母吵架"

### 3. 内容价值
- 每周推送一篇亲子沟通文章
- 分享其他用户的成功故事
- 节日提醒："母亲节快到了，要不要给妈妈写张卡片？"
